venvより、pipenvを使う

## インストール

$ pip install pipenv

## 初期化

$ pipenv --python 3.8

## パッケージのインストール

$ pipenv install numpy    # numpyをインストールする例です

## requirements.txtからのインストール

$ pipenv install -r ./requirements.txt

## すでにPipfileがある場合

$ pipenv install
