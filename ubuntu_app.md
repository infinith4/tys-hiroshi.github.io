## aliasは専用の設定ファイルが用意されている

create .bash_aliases

```
alias cat='cat -n'
alias emacs='emacs -nw'
alias python='/usr/bin/python3'
alias pip='/usr/bin/pip3'
```

```
source ~/.bashrc
```
